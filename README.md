# NgemilPedia

Pusat Aneka Cemilan Lezat! NgemilPedia adalah platform katalog digital untuk produk cemilan rumahan, mulai dari frozen food (Nugget & Sosis) hingga berbagai varian Kue Kering premium.

## Fitur Utama
- **Desain Modern & Responsif**: Tampilan bersih dengan estetika premium menggunakan font *Outfit* dari Google Fonts.
- **Katalog Produk Terstruktur**: Layout grid yang memudahkan pelanggan melihat detail produk, berat (gramasi), dan harga.
- **Integrasi WhatsApp**: Memudahkan pemesanan langsung melalui tautan WhatsApp yang tersedia di setiap bagian penting.
- **Optimasi Performa**: Penggunaan gambar `.webp` dengan *lazy loading* untuk kecepatan akses yang maksimal.

## Teknologi yang Digunakan
- **HTML5**: Struktur semantik untuk SEO dan aksesibilitas.
- **Vanilla CSS**: Desain kustom dengan variabel CSS, Glassmorphism, dan animasi halus.
- **JavaScript**: Fungsionalitas interaktif dasar untuk pengalaman pengguna.

## Cara Menggunakan
Cukup buka `index.html` di browser pilihan Anda untuk melihat katalog. Untuk melakukan pemesanan, klik tombol "Pesan via WhatsApp".
